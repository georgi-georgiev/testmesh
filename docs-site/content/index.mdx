## What is TestMesh?

TestMesh is an open-source test automation platform designed for teams that need to build, manage, and execute tests at scale. It provides a visual flow editor for creating test workflows, a plugin system for extending functionality, and powerful scheduling capabilities.

## Quick Start

Get up and running with TestMesh in minutes:

```bash
# Clone the repository
git clone https://github.com/yourusername/testmesh.git

# Start with Docker Compose
docker-compose up -d

# Open the web UI
open http://localhost:3000
```

Ready to dive deeper? Check out the [Getting Started guide](/docs/getting-started).
